Educator's Guide Template

Standardized lesson planning framework for all Curriculum Lab programs. Professional development tool.

Overview

This template ensures every Curriculum Lab lesson follows proven educational structure while allowing instructor flexibility. Designed for consistency across diverse programs while maintaining creative teaching approaches.

Learning Outcomes

  • Write clear, measurable lesson objectives
  • Structure 60-minute experiential lessons
  • Design inclusive differentiation strategies
  • Create safety protocols and material lists
  • Embed assessment and reflection moments

Core Learning Themes

  • Experiential Arc: Hook → Challenge → Reflect
  • Universal Design: Multiple access points
  • Safety Integration: Proactive protocols
  • Assessment Embedded: Formative throughout

Engagement Design

"Ready-to-teach in 15 minutes" fillable template. Visual lesson flow diagram helps new instructors see timing relationships. Progressive complexity guides scaffold lesson writing from simple to multi-day units. Peer review checklist builds confidence.
